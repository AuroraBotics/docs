A periodic frame containing generic data about the device and it's current state, including active and sticky faults.

| Property     | Value           |
| ------------ | --------------- |
| API Class    | `000000`        |
| API Index    | `0000`          |
| Data Length  | 8 Bytes         |
| TX Direction | Device -> Robot | 

| Data                   | Start Bit | Data Length | Description                         |
| ---------------------- | --------- | ----------- | ----------------------------------- |
| `Active Fault Bitmask` | 0         | 1 Byte      | Bitmask of active [[Faults]].       |
| `Sticky Fault Bitmask` | 8         | 1 Byte      | Bitmask of sticky [[Faults]].       |
| `Temperature`          | 16        | 1 Byte      | Current temperature of device in Â°C |
| `Reserved`             | 24        | 5 Bytes     | Reserved for future implementation. |
