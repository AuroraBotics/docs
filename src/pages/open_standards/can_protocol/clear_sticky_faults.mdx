---
tag: CAN-API
alias: clear_sticky_faults
---

Clears all sticky faults, however, if there is an active fault, the sticky fault will persist. Check [[Faults]] for a list of faults that Aurora can raise.

| Property     | Value           |
| ------------ | --------------- |
| API Class    | `000000`        |
| API Index    | `1000`          |
| Data Length  | 0 Bytes         |
| TX Direction | Robot -> Device |

There is no data transmitted.
